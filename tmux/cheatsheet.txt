  tmux   —   prefix = Ctrl-Space
  (tap Ctrl-Space first, then the command key)


  WINDOWS    (like browser tabs)

    c           new window
    n  /  p     next / previous
    1 .. 9      jump to window N
    ,           rename current window
    &           close window  (asks to confirm)


  PANES      (splits inside a window)

    |           split vertically   (side-by-side)
    -           split horizontally (top + bottom)
    ← ↓ ↑ →     move between panes
    z           zoom current pane  (toggle full screen)
    x           close pane
    space       cycle pane layouts (pretty rearrange)


  SESSION    (the whole tmux workspace)

    d           detach (leave it running, come back later)
    s           switch to another session
    $           rename session


  HELP

    h           show this cheat sheet
    ?           list every key binding
    r           reload tmux config


  Scrollback: just use the mouse wheel. Copy-mode ( [ ) is only
  for searching scrollback or copying text across panes — you
  probably won't need it day to day.
