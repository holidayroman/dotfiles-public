── tmux cheat sheet ──────────────────────────────────────
Prefix is Ctrl-a   (press Ctrl-a Ctrl-a to send a literal
                    Ctrl-a to vim / readline)

  SESSIONS
    tmux ls          list sessions (from shell, no prefix)
    Ctrl-a d         detach (session keeps running)
    Ctrl-a s         switch/choose session
    Ctrl-a $         rename session

  WINDOWS (like tabs)
    Ctrl-a c         new window
    Ctrl-a n / p     next / previous window
    Ctrl-a 1..9      jump to window N
    Ctrl-a ,         rename current window
    Ctrl-a &         kill current window (asks to confirm)

  PANES (splits within a window)
    Ctrl-a |         split vertically
    Ctrl-a -         split horizontally
    Ctrl-a arrow     move between panes
    Ctrl-a z         zoom current pane (toggle full-screen)
    Ctrl-a x         kill current pane

  COPY / SCROLLBACK
    Ctrl-a [         enter copy-mode (vi keys)
      v              start selecting
      y              yank selection (exits copy-mode)
      q              quit copy-mode
    Mouse wheel      scroll (mouse mode is on)

  MISC
    Ctrl-a r         reload ~/.tmux.conf
    Ctrl-a h         show this cheat sheet
    Ctrl-a ?         list ALL key bindings
